Conectar con Google Sheets

Esta app usa Google Sheets como base de datos. Necesitás crear un Apps Script y pegar la URL del Web App aquí. Ver instrucciones

📋 Instrucciones

  1. Abrí Google Sheets y creá una hoja nueva.
  2. Andá a Extensiones → Apps Script.
  3. Borrá el código existente y pegá el contenido del archivo google-apps-script/Code.gs.
  4. Reemplazá TU_SPREADSHEET_ID con el ID de tu hoja (lo ves en la URL).
  5. Guardá y hacé clic en Implementar → Nueva implementación.
  6. Elegí tipo Aplicación web, ejecutar como Yo, acceso Cualquier usuario.
  7. Copiá la URL que te da y pegala aquí.
