##APP de Gastos propios##
#Definiciones Generales.
- App para registrar gastos propios, sin necesidad de subirla a google play. Si que pueda ejecutarla en un navegador y como pwa.
- Realizarla en Java Script 
- Que sea PWA
- La Base de Datos es un google Sheet
- Podemos tener mejoras a futuros

##Secciones:

Secciones:
1- Apertura Periodo (Admin)
 - Que pueda abrir Ejercicios a futuros, el 2026 que ya se encuentre abierto.
2- Tipos de Ingreso (Admin)
 - Son los tipos de Ingreso permitidos.
3- Formas de Pago (Admin)
 - Distinas formas de pago.
4- Gastos 
  * Poder registrar cada gasto diario, por fecha, importe, Forma Pago , Cuenta origen, categoria, Si es un Servicio o no.
5- Ingresos
 - Poder realizar ingresos a una Cuenta.
 - Poder realizar transferencias entre cuentas.
 - Poder realizar compra de USD con dinero en cuenta a una cuenta en USD
6- Realizar un resumen de mis cuentas.
 - Quiero saber dinero disponible en cada cuenta (Ingreso - Egreso)
 - Si se realizo una transferencia entre cuenta dar de baja el dinero de una cuenta y sumarla en la otra cuenta.
 - Si se realiza un gasto realizar el egreso de la cuenta que se seleccione.
 - Poder agregar otras cuentas si tengo la necesidad.
 - Dashboard:
    - x categoria con filtro de mes y ejercicio.
